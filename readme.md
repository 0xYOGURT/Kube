#Kube – OpenGL Rotating Cube Demo

This project is a simple **OpenGL 3.3** demo using **GLFW** and **GLAD** that renders a colorful rotating cube with a glow effect based on time.  

## Features
- Rotating 3D cube with vertex colors
- Dynamic glow effect using fragment shader
- Basic matrix math implemented manually (model, view, projection)
- Uses modern OpenGL (core profile)

## Requirements
- Windows 10/11
- **MinGW-w64** or **Visual Studio**
- [GLFW](https://www.glfw.org/) library
- [GLAD](https://glad.dav1d.de/) OpenGL loader
